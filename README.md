# Locadora de Veículos - SQL

Projeto acadêmico desenvolvido para praticar modelagem de banco de dados relacional e consultas SQL.

## 📌 Estrutura do Projeto

- `database/schema.sql`  
  Criação do banco de dados e tabelas

- `database/inserts.sql`  
  Inserção de dados fictícios para testes

- `queries/`  
  Consultas SQL com foco em análise de dados e extração de informações relevantes

## 📊 Consultas implementadas

- Listagem de manutenções realizadas
- Cálculo do valor total arrecadado
- Quantidade de locações por veículo (veículos mais alugados)
- Identificação de clientes com pagamentos pendentes
- Receita gerada por veículo

## 🛠 Tecnologias

- SQL (MySQL)

## ▶️ Como executar

1. Executar o arquivo `schema.sql`
2. Executar o arquivo `inserts.sql`
3. Executar as consultas da pasta `queries/`

## ⚠️ Observação

Os dados utilizados neste projeto são fictícios e foram utilizados apenas para fins acadêmicos.
